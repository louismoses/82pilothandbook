# 1. Flight Plan (FP)

## 1.1 Filing Requirements

- File 30â€“60 minutes before flight.
- Ensure Necessary documents are filled.
    - Flight Plan
    - Aircraft Security Clearance
    - CAAP Manifest Form

- Scan filename format:

> _FLIGHT PLAN (Ident TimeZulu Year Month Date).pdf_

Example:  
_FLIGHT PLAN (RP-C3519 0000Z 2026 February 06).pdf_

## 1.2 Email Format

**To**

```
mactanfss @caap.gov.ph
```

**CC**

- Capt. Buscas
- Sir. Ian Nucum
- Your Captain

**Subject**

```
FLIGHT PLAN (RP-C3519 0000Z 2026 February 06)
```

**Message Template**

```
This is Cadet [Name] from AAG Bantayan Station.
Attached is the Flight Plan for RP-C3519 0000Z 2026 February 06.

Thank you.
```

## 1.3 Confirmation

After approximately 10 minutes, call radio to confirm.
0933-934-335\_

```
CDT: "Mactan Radio, Cadet [Name] from AAG Bantayan Station, confirming flight plan for RP-C3519 time 0000Z."
```

```

```

# 4. Departure

```
CDT: "Bantayan Advisory, [Ident], holding short south taxiway, query traffic on the pattern"

CDT: "Negative traffic received, [Ident], lining up runway [34] Bantayan"

CDT: "Bantayan Advisory, [Ident], rolling [34] Bantayan"
```

## 4.1 Local

```
CDT: "[Ident], airborne [34] Bantayan [Touch and go]"

CDT: "[Ident], right [crosswind/downwind/base] [34] Bantayan [touch and go]"

CDT: "[Ident], [final] [34] Bantayan, [full stop/touch and go]"
```

## 4.2 Non-local

```
CDT: "Bantayan Advisory, [Ident], airborne [34], right downwind departure, destination [Ormoc], will report living the pattern "

CDT: "Bantayan Advisory, [Ident], living the pattern passing 1000ft, climbing [2500ft] destination [Ormoc] via [San Remigio] "

```

# 5. Arrival

```
CDT: "Bantayan Advisory, [Ident], over [Madridejos], [1500ft], proceeding west of station"

CDT: "Bantayan Advisory, [Ident], west of station, [1500ft], proceeding overhead station, query traffic over the station"

CDT: "Negative traffic received, [Ident] will proceed over the station"

CDT: "Bantayan Advisory, [Ident] overhead station [1500ft], query traffic on the pattern"

CDT: "Negative traffic received, [Ident] will join right mid-downwind"

CDT: "[Ident] established right mid-downwind [34] Bantayan"
```

```
CDT: "[Ident], landed [34] Bantayan, will make 180 and vacate North"

CDT: "[Ident], Runway clear, closing flight plan. Thank you and Good day"
```
